# 🛍️ Front-end de E-commerce – Loja Real

Este repositório contém o **front-end de um site de e-commerce** desenvolvido para uma loja real. O projeto foi estruturado para oferecer uma experiência moderna, responsiva e intuitiva, permitindo que clientes naveguem, visualizem produtos e realizem compras com facilidade.

---

## 📂 Estrutura do Projeto

- **index.html** → Página inicial do e-commerce, estruturada para navegação clara e responsiva.  
- **css/** → Estilos customizados para layout, tipografia e responsividade.  
- **js/** → Lógica de autenticação, gerenciamento de produtos e interatividade.  
- **assets/** → Imagens e recursos visuais utilizados no site.  
- **README.md** → Documentação do projeto.  

---

## 🎯 Objetivos

- Criar uma interface **responsiva e amigável** para usuários.  
- Implementar **funcionalidades essenciais** de um e-commerce (menu, autenticação, exibição de produtos).  
- Garantir **boa performance** e compatibilidade em diferentes dispositivos.  

---

## 🚀 Funcionalidades

- **Menu responsivo** para navegação em dispositivos móveis.  
- **Autenticação de usuários** integrada ao front-end.  
- **Gerenciamento de produtos** com lógica implementada em JavaScript.
- **Opção de admin** para fazer modificações de maneira intuitiva, sem precisar mecher diretamente com o código.
- Estrutura modular para facilitar manutenção e evolução futura.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** → Estrutura semântica e acessível.  
- **CSS3** → Estilização responsiva e moderna.  
- **JavaScript** → Lógica de autenticação, interatividade e manipulação de produtos.
